# urtica-sdr-analysis
# Identifying the sex-determining region of Urtica dioica

Analysis code and processed data accompanying an MSc dissertation submitted in 2026.

This repository will be populated with code, processed data tables, and figures shortly. The reorganisation of working files is in progress.

## Planned contents

- `scripts/` — custom analysis scripts (Python, R, shell)
- `data/` — summary tables underlying dissertation figures
- `docs/` — methods mapping, analysis walkthrough, tool versions
- `figures/` — main and supplementary figure files
